What is weather?

weather is a command line tool for retrieving the weather forecast for your current location or a given city.

How do I get weather?

mac/Linux Windows
  • Download the tar.xz archive to your machine
  • Extract the archive
  • Move file:
    (sudo) mv <path_to_weather> /usr/local/bin/weather
  • Run weather
  • Download the zip archive to your machine
  • Extract the archive
  • Move file to a folder which is part of your PATH
  • Run weather
